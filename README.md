# Bank Data Platform

Projet de plateforme de données simulant un pipeline bancaire.

## Architecture du projet
- **ingestion/** : ingestion des transactions
- **airflow/** : orchestration du pipeline (DAGs)
- **dbt_project/** : transformation des données (modèles dbt)
- **api/** : exposition des données via une API

## Technologies utilisées
- Python
- SQL
- Airflow
- dbt
- FastAPI
- Git / GitHub
